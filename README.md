# vue-ts-started

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### lint 체크 및 수정
```
vue-cli-service lint
```

lls, transpiling JSX)?

in production)
**Yes**

`?` Pick additional lint features:

`?` Where do you prefer placing config for Babel, ESLint, etc.?
